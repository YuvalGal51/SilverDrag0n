Welcome to SilverDrag0n.

SilverDrag0n is a java application made for pentesters and people who learn cyber & social engineering like me. It includes powerfull tools like apache2 & beef-xss framework handling for social engineering html.

Requirements

Follow this two steps to get the program to run properly:

1. Use kali linux 2.0 and above to run the program.

2.Run it by navigating to it's location in the terminal and typing "java -jar SilverDrag0n.jar".
